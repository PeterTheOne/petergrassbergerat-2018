Privacy

This website uses Piwik to track user visits. The Piwik installation runs on the same server and tracking data is not given to a third party. As recommended by wirspeichernnicht.de with ip_address_mask_length = 2 only two bytes of the IP address are saved and the browser option Do Not Track is respected. To keep track of repeated visitors Cookies are used. If you don't want to be tracked activate the Do Not Track option in your Browser, block javascript or the subdomain piwik.petergrassberger.com or opt-out below.
