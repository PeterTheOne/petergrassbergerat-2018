Privatsphäre

Diese Webseite verwendet die Software Piwik um Statistiken von Seitenzugriffen aufzuzeichnen. Die Piwik Installation wird auf dem gleichen Server betrieben und die Daten werden nicht an Dritte weitergegeben. Nach Empfehlung von wirspeichernnicht.de werden mit ip_address_mask_length = 2 nur zwei bytes, also ein kleiner Teil der IP-Adresse gespeichert, au\erdem wird die Browser Option Do Not Track respektiert. Um nach zu verfolgen welche Nutzer diese Webseite öfter besuchen werden sogenannte Cookies verwendet. Wenn man nicht möchte, dass über einen Besucherstatistiken aufgezeichnet werden, kann man diese Aufzeichnung übergehen indem man die Do Not Track Option im Browser aktiviert, javascript oder die Subdomain piwik.petergrassberger.com blockiert oder sich unter abmeldet.
