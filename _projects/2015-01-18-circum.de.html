Cicum ist ein web basierter Panorama Viewer mit integrierten HTML-Elementen wie Text, Bild oder Video. In Circum selbst können super einfach Interaktive Entdeckungsreisen erstellt werden, auch vom Smartphone aus. Die Entwicklung habe ich zuerst für websafari GmbH gemacht, später dann zusammen mit Erfindler GmbH.

Entwicklung und Technik

Bei diesem Projekt war mein Fokus sowohl am Backend als auch am Frontend, im Web Design hatte ich immer wieder auch Unterstützung. Im Backend liegt eine SQLite Datenbank, die über eine mit dem Slim Framework umgesetzte REST API ansprechbar ist, das Frontend basiert auf Backbone.js, Underscore.js und Require.js.
